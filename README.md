# global-gender-balance-analysis
This project analyses global gender balance trends using population data across regions, income groups, and time.  The goal is to provide insights that support policy decisions and promote gender equality. 

## Problem Statement
- How has gender balance changed over time?
- Which regions show the greatest imbalance?
- What trends can be expected in the future?

## Tools & Technologies
- Power BI
- DAX
- Data Modelling
- Data Cleaning (Power Query)

## Key Steps
1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Data Modelling
4. Dashboard Development

## Key Insights
- Gender ratios are relatively stable globally
- Regional disparities still exist
- Income levels influence gender balance trends

## Impact
- Supports policy and planning decisions
- Helps identify gender inequality patterns
- Improves accessibility of demographic insights

## How to Use
Download the Power BI file from the dashboard folder and open in Power BI Desktop
